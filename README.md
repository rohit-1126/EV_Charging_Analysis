# EV_Charging_Data_Analysis

## Objectives Achieved
- Cleaned and prepared the EV charging dataset.
- Performed descriptive statistical analysis.
- Applied Excel functions (logical, lookup, text, math, statistical).
- Performed industry-focused analysis of EV charging usage.
- Use Pivot Tables to summarize charging behavior.
- Create charts to visualize insights.
- Build an interactive dashboard summarizing operational insights.


## Tools Used
- Google Sheets & Functions
- Pivot Tables
- Charts

## Key Insights
- Fast charging stations were used most frequently.
- 3W EV Type Vehicle consume more energy.
- Kolkata City is leading in revenue contribution with a ~9% higher than average sales.
- Energy Consumption has negative slope in comparing 2023-24 trend.
